# @osmn-byhn/storybook-docs

Documentation and UI component playground for the `@osmn-byhn` ecosystem.

## Usage

Start the Storybook development server:

```bash
pnpm run storybook
```

Build the static Storybook site:

```bash
pnpm run build-storybook
```

## Structure

- **stories/**: Contains the stories defining the component states and documentation pages.
